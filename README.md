## Requisitos

- Possuir PHP instalado
- Possuir Composer instalado

## Instalar as dependências com o comando
- php artisan breeze:install
- npm install


## Realizar o build das dependências Javascript:
- npm run dev

## Realizar as migrations:
- php artisan migrate

## Executar o projeto:
- php artisan serve
